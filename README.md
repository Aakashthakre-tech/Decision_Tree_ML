# 🌳 Decision Tree Classification Project

This project demonstrates a complete Machine Learning workflow using the **Decision Tree algorithm** with industry-standard libraries.

It includes data preprocessing, feature engineering, model training, evaluation, and performance analysis.

---

## 🚀 Project Highlights

- Data cleaning & preprocessing
- Handling categorical variables (Encoding)
- Feature scaling (if required)
- Train-Test split
- Decision Tree model training
- Model evaluation using:
  - Accuracy Score
  - Confusion Matrix
  - Classification Report
- Basic hyperparameter tuning

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

---

## 📂 Workflow

1. Load dataset
2. Perform exploratory data analysis (EDA)
3. Encode categorical features
4. Split data into training and testing sets
5. Train Decision Tree classifier
6. Evaluate model performance
7. Analyze results

---

## 📊 Model Used

- `DecisionTree` from scikit-learn

---

## 🎯 Objective

To understand how Decision Trees are applied in real-world ML pipelines using professional tools and best practices.
